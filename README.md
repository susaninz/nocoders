# 🧠 Dev Knowledge Base

Персональная база знаний по разработке программного обеспечения.

## 📚 Содержание

- **Python** — практики разработки, async, SQLAlchemy, тестирование
- **Архитектура** — Clean Architecture, паттерны, State Machine
- **Telegram боты** — разработка ботов на python-telegram-bot
- **Инструменты** — Git, Docker, CI/CD

## 🚀 Быстрый старт

### Локальный просмотр

```bash
# Установить MkDocs
pip install mkdocs-material

# Запустить локальный сервер
mkdocs serve

# Открыть http://localhost:8000
```

### Деплой на GitHub Pages

```bash
mkdocs gh-deploy
```

## 🛠️ Структура

```
dev-knowledge-base/
├── docs/
│   ├── index.md              # Главная страница
│   ├── python/               # Python раздел
│   ├── architecture/         # Архитектура
│   ├── telegram-bots/        # Telegram боты
│   ├── tools/                # Инструменты

```

## ✨ Возможности

- 🔍 Полнотекстовый поиск
- 🌙 Тёмная/светлая тема
- 📱 Адаптивный дизайн
- 📋 Копирование кода одним кликом
- 🏷️ Теги и категории

## 📝 Добавление контента

1. Создай markdown файл в соответствующей директории
2. Добавь его в `nav` секцию в `mkdocs.yml`
3. Запусти `mkdocs serve` для проверки
4. Закоммить и запушить
